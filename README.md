## Requirements

- macOS 11.0 or later
- Xcode 14.0 or later
- Swift 5.0 or later

## Usage

After launching, PowerMom will appear in your menu bar showing:
- Battery percentage
- Current power source
- Dynamic battery icon that changes based on:
  - Battery level
  - Charging status

Click the menu bar icon to view detailed information and access settings.
